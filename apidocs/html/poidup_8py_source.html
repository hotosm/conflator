19 # Note that this program is not designed to be fast, it's designed
20 # to be complete. It's a simple brute force algorithym. The input
21 # data file has to already be converted to OSM syntax of course.
22 # If a tag from the input file is found in any OSM features within
23 # a short distance, then the feature is flagged as a possible
24 # duplicate. Anything not flagged is likely new data.
25 #
26 # The lack of performance is tolerable with smaller input datasets,
27 # but obviously this won't scale. The conflator.py program is
28 # designed for larger datasets, so if that becomes necessary.
